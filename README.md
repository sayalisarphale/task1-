# Data Cleaning & Visualization Project

## Overview
This project demonstrates the process of cleaning and visualizing a raw dataset using Python.

## Features
- Handle missing values
- Remove duplicate records
- Detect and treat outliers
- Perform data visualization
- Generate cleaned dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Files
- data_cleaning.py
- dataset.csv
- cleaned_dataset.csv
- output_images/
